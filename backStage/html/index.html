February 22, 2020 | by studyHooligan | 0 Comments

The first time to set up a home page.The background uses NodeJS+express framework for services, and the front end uses pure HTML5 technology.The reason for abandoning mainstream front-end frameworks such as Vue is that it is said that HTML5 is great and supports some exciting new features, such as video tags.

Some interesting new features in HTML5:
  • Canvas element for painting
  • Video and audio elements for media playback
  • Better support for local offline storage
  • New special content elements, such as article, footer, header, nav, section
  • New form controls, such as calendar, date, time, email, url, and search
